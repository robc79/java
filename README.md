# README

This repository is for learning Java and its ecosystem, and contains solutions for the following:

## hello_world

The ubiquitous starting point for any language.

## rps

An OOP implementation of the game Rock, Paper, Scissors to get used to the Java
syntax.

## wc

A simple copy of part of the Unix `wc` command.

## gcp

An naive implementation of the saturation degree heuristic for solving the graph
colouring problem. Using Maven for the first time.

## rosalind

CLI utility that implements some of the bioinformatics functions outlined at
the [Rosalind](https://rosalind.info/problems/locations/) project website.
